## Installation

1. Install NPM packages.

```sh
npm install
```

2. Create .env file in the root folder.

```sh
touch .env
```

3. Enter your information in .env file

```env
REACT_APP_GITHUB_PAT="someGithubPersonalAccessTokenFromDeveloperSettings"
```
